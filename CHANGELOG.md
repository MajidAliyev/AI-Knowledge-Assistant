# Changelog

## [1.0.0] - 2024-11-04

Initial release.

### What's Included

- Document ingestion (PDF, Markdown, Text, Email)
- RAG pipeline with Ollama
- React chat interface
- Source citations
- Dark/light theme
- Document upload UI
- Semantic search
- Local vector database (ChromaDB)
- Fast model (phi3) as default

### Features

- Everything runs locally
- No external APIs
- Privacy-first
- Modern UI
- Easy setup scripts
